      # About this project:
      
   
      This project was developed for NASA's space apps challenge
      by a team of three students and developers from India.
      Project Chandra focuses on making the logging system better 
      and easier to use for NASA's flight controllers.
      
      
      # Other Info:
      
      ## Logging system:
      Our logging system is made with HTML, CSS, Python and TinyDB      
      
      ## Lunar surface mapping:
      Try out our 3D model of the moon, explore and find safe landing spots.
